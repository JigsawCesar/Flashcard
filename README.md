# Flashcards

_Sistema de Gerenciamento de Flashcards interativo no terminal em JS_

![Versão](https://img.shields.io/badge/version-1.0.0-blue)

## Funcionalidades

- Criação de Baralhos e Flashcards.
- Listar Baralho, Flashcard e listar por Baralho.
- Atualizar Baralho e Flashcard.
- Deletar Baralho pelo seu ID com seus respectivos Flashcards e remover Flashcard.
- Buscar flashcards baseado no ID do seu Baralho.
- Interface interativa via terminal.
- Armazenamento local dos dados nos arquivos: baralho.js e flashcard.js

## Tecnologias

- ![Node.js](https://img.shields.io/badge/Node.js-339933?logo=node.js&logoColor=white)
- JavaScript
- npm
- Prompt Sync

## Instalação

```bash
git clone https://github.com/JigsawCesar/Flashcard.git
cd Flashcards
npm install prompt-sync
node index.js
```
